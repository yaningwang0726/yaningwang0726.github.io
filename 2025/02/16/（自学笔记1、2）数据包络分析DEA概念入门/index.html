（自学笔记1）数据包络分析DEA概念入门

前提概念

技术效率：指一个生产单元的生产过程达到该行业技术水平的程度，即一个生产单元技术水平的高低。技术效率可以从投入和产出两个角度进行衡量，在投入既定的情况下，技术效率由产出最大化程度衡量（产出导向型）；在产出既定的情况下，技术效率由投入最小化程度来衡量（投入导向型）。

技术效率可以通过产出/投入的比值进行测量。即当生产过程中仅涉及一种产出和一种投入的时候，可以通过产出/投入（每消耗一个单位的投入所生产的产品数量），反映各生产单元技术效率的高低。各单元产出/投入的值除以所有单位中的最大值，就能得到标准化后的产出/投入（在0-1范围内），从而更好地反映评价单元与其他单元之间的技术差距。

这种情况只涉及到单效率、单产出的情况。但现实的生产过程往往涉及多项投入和产出，因此可以通过对不同的投入和产出指标赋予一定的权重，然后计算出加权产出/加权投入的比值，可以作为反映技术效率的指数。

那么接下来要考虑如何确定权重？一种方式是通过主观评价，如：专家打分法等获得固定权重；另一种则是通过数据本身获得投入和产出的权重，DEA便是采用了这种方法。

DEA的方法说明和模型解释

DEA方法通过对决策单元（DecisionMaking Units,DMU），即同类型及条件下的“生产部门”投入与产出指标的评估，进行量化的效率值计算以及有效性评价。

DEA的优势：能够对多输入多输出问题进行有效性评价，无需权重假设、无需数据量纲化、便于进行动态分析。

DEA的模型：CCR、BCC、SBM、TFP等

CCR模型由Charnes、Cooper和Rhodes于1978年提出，可以计算规模报酬不变（Constant return to scale，CRS）情况下的资源配置效率。BCC模型则是前两位加上Banker在1984年对CCR模型的拓展，即规模报酬可变（Variable return to scale，VRS）。

【什么是规模报酬：规模报酬又叫规模收益，指在技术水平和要素价格不变的情况下，所有要素按照同一比例变动时，收入也同比例变动的情况。比如你的资本投入增加了100%，那么产量（收益）也会增加100%。大于100%则称为规模报酬递增，反之为规模报酬递减。】

从这个角度上来说，CCR模型也可以叫做CRS径向DEA模型，BCC模型可以叫做VRS径向DEA模型。径向的含义指无效率的测量方式为投入能够等比例减少的程度或者产出能够等比例增加的程度。（这个概念不理解也无所谓）

传统的CCR和BCC模型只能横向比较决策单元在同一时点的生产效率【截面数据】，而DEA-Malmquist指数模型可以测度决策单元在不同时期效率的动态变化，因此可以分析【面板数据】，更具广泛的应用性。

前文提道DEA可根据投入或者产出既定分为投入导向型【不减少产出的条件下，达到技术有效时投入可以减少的程度】和产出导向型【在不增加投入的情况下，达到有效产出应该增加的程度】。在CRS径向中，二者的效率值是一样的，但在VRS径向中，二者效率值是不等的。

CRS计算出的效率值叫做技术效率（TE，technical efficiency），其中包含了规模效率（SE，scale efficiency），并不是纯技术效率（PTE，pure technical efficiency）。VRS计算出来的才是纯技术效率，因此可以算出SE=TE/PTE

在软件中操作的名词说明

技术效率 技术进步 纯技术进步 规模效率 全要素生产率
TECH TECHCH PECH SECH TFPCH
反映对现有资源有效利用的能力，体现决策单元组织管理水平的变化 测量技术效率的进步情况 反映管理、制度体制等因素对生产率的影响 反映现有规模与最优规模的差异

在规模报酬不变情况下：TFP是技术效率变动指数TECH和技术进步变动指数TECHCH的乘积

在规模报酬可变情况下：TECH可分解为纯技术效率变动指数PECH和规模效率变动指数SECH的乘积。因此TFP=PECH×SECH×TECHCH

若TFP＞1，则表示从第t期到t+1期的全要素生产率提高，反之下降。PECH、SECH、TECHCH的解释同理，若有一项大于1，则表明该要素是全要素生产率增长的源泉，反之则导致了它的下降。

SBM超效率模型

在DEA模型的结果中，通常会出现多个DMU被评价为有效的情况，DEA模型得出的效率值最大为1，有效DMU的效率值相同，这些有效DMU的效率高低无法进一步区分。为解决这一问题，Andersen和Petersen（1993）提出了对有效DMU区分其有效程度的方法，这种方法被称为“超效率”模型（super efficiency model），SBM便是超效率模型的一种。

此外，DEA作为一种线性规划模型，其根本思想是用最小的投入获得最大产出，但生产系统中往往包含好产出和坏产出（期望产出和非期望产出），【假如两个国家使用相同的投入，获得了相同产出的GDP，但二者产生的污染不同，这就表明二者的效率不同】
